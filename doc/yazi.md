# Yazi Cheat Sheet

## Quick Keys

| Key | Action |
|-----|--------|
| `.` | Toggle hidden files |
| `q` | Quit |
| `enter` | Open file/folder |
| `h` / `l` | Navigate left/right (parent/child) |
| `j` / `k` | Navigate up/down |
| `y` | Yank (copy) |
| `p` | Paste |
| `d` | Delete |
| `r` | Rename |
| `/` | Search |
| `Space` | Select file |
| `~` | Go to home |

## Config

Config location: `~/.config/yazi/yazi.toml`

```toml
[manager]
show_hidden = true
```
